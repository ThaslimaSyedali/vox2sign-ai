   Vox2Sign AI - Speech to Sign Language Converter

An AI-powered real-time system that converts spoken speech into 
sign language gestures, built to aid deaf and hard-of-hearing communities.

 🚀 Features
- Real-time speech to sign language conversion
- Google Speech API integration
- Sequential gesture video playback
- Noise adjustment and error handling
- Accessible GUI with cross-platform support

 🛠️ Tech Stack
- Language: Python
- Libraries:OpenCV, SpeechRecognition, Tkinter, PIL/Pillow
- API: Google Speech API
- Tools: Git, VS Code

 ⚠️ Note
Full gesture dataset not included due to file size limitations.
Main application file: `vox2sign.py`
